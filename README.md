## Landing Page menu 

Website menu digital sederhana untuk restoran/warung makan dengan tampilan modern, tab kategori (Makanan, Minuman, Cemilan), fitur pencarian real-time, dan data terpisah di file JSON.

## Fitur

- Desain responsif (mobile-first friendly)
- 3 kategori menu: Makanan · Minuman · Cemilan
- Tab switching dengan animasi halus
- Pencarian menu secara real-time (live search)
- Harga diformat dengan pemisah ribuan (toLocaleString)
- Data menu dipisah ke file `data.json` (mudah di-update)
- Menggunakan font Poppins dari Google Fonts
- Hover effect & shadow modern

## Teknologi yang Digunakan

- HTML5
- CSS3 (Flexbox + Grid + Gradient)
- Vanilla JavaScript (tanpa framework)
- Fetch API untuk load data JSON
- Google Fonts
